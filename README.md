#  Arch Sway Dotfiles

A high-performance, keyboard-centric environment tailored for AMD hardware. This configuration prioritizes a "blazingly fast" workflow using a Tokyonight aesthetic and strict workspace segmentation.

##  Core Stack

- **Window Manager:** Sway (Wayland)
- **Shell:** Zsh powered by Starship prompt
- **Text Editor:** Neovim (Tokyonight Storm)
- **File Management:** Yazi + `udiskie` (Auto-mount mapped to `~/usb`)
- **Menu/Launcher:** Wofi
- **Status Line:** Waybar
- **Notifications:** Mako
- **Terminal:** Ghostty (GPU accelerated)

##  Hardware & Performance

- **CPU:** AMD Ryzen 7 5825u
- **GPU:** Radeon (Optimized with `vulkan-radeon` & `radeontop`)
- **Font:** JetBrainsMono Nerd Font
- **Automount:** D-Bus managed storage for seamless hot-swapping
